# 道德经--第七章

> **Date / 日期**: 2026-03-29

---

**原文：**

```
天长，地久。

天地之所以能长且久者，以其不自生也，故能长生。

是以圣人后其身而身先，外其身而身存。

非以其无私邪？故能成其私。
```

**译文：**

```
天长地久，

天地所以能长久存在，是因为它们不为了自己的生存而自然地运行着，所以能够长久生存。

因此，有道的统治者能够保持谦让不争，反而能在众人之中领先；将自己置之度外，反而能够保全自身。

这不正是因为他的无私吗？所以才能成就自己。
```

**个人体会：**

天长地久是因为天地不为自己生存，不可以谋求自我存续，才能永恒长存。

天地跳脱出自己的身份，致力于孕育万物，自己也会生生不息；

统治者跳出参与分配的角色，重点在如何公平地推动财富分配，国家治理就能长治久安。

如果既当裁判员，又当运动员，自私的行为迟早会反噬自身。

就像最近有个段子，问懂王美伊战争是赢了还是输了，懂王说：赚了。

财聚人散，财散人聚，最朴素的道理，一句话就能讲清楚。

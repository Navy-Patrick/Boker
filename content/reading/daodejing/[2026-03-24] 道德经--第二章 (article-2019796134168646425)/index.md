# 道德经--第二章

> **Date / 日期**: 2026-03-24

---

**原文：**

```
天下皆知美之为美，斯恶已；

皆知善之为善，斯不善已。

故有无相生，难易相成，

长短相较，高下相倾，

音声相和，前后相随。

是以圣人处无为之事，行不言之教。

万物作焉而不辞，生而不有，为而不恃，功成而弗居。

夫唯弗居，是以不去。
```

**译文：**

```
天下人都知道美之为美，那是有丑的存在。

都知道善之为善，那是因为有恶的存在。

所以有和无互相转化，难和易互相形成；

长和短互相显现，高和下互相充实；

音与声互相协同，前和后互相紧随——这是永恒的规律。

因此圣人用无为的方式来行事，用不言的方式来教化。

听任万物自然兴起而不刻意去改变，有所作为但不加强加个人意志，取得成就而不自居。

正因为不居功，反而不会失去。
```

**个人理解：**

如果天下人都去追求美，用一个标准来规范美，那么美的另一端，丑的概念也就被人为创造出来；

人之所以会产生比较攀比的心态，从而产生羡慕嫉妒等的情绪，就是因为有一把尺子来统一标准，善恶也是如此，脱离这个标准，人就无法衡量评判。

自然是本无绝对的美丑善恶之分，是因人的“”和“刻意标榜”产生，制造了对立与偏执。

客观事物是对立统一相互转化的，没有长，就没有短；没有高，也就没有下；没有前，就没有后。世间对立的事物都是相互成就，缺一不可的，没有绝对的好坏，高低，长短。

圣人处无为之事，不是躺平不作为，而是不违背自然规律，顺应天时，平常心去做事，有为之实就随之而来。

圣人行不言之教，不是靠说教，而是以身作则，潜移默化影响身边之人，不做望子成龙的事情。

万物自然生长、兴起、发展，不去干预、阻碍、控制，帮助给予滋润却不占为己有，有所作为但不自以为是恃才傲物，功成名就但不自视甚高，把功劳都归在自己身上。

正因为不居功，所以功绩才不会消失，不会被磨灭。

如果一个人一旦居功，就会执着于“这功劳是我的”，从而沾沾自喜，会虚荣想要炫耀，而炫耀的本质是人无我有，引来自己的盲目自大和他人的嫉妒争夺，功劳也会因为自己的执念而失去。

如果不居功，做事的心态没有受到影响，本就是顺其自然，无心为私，功绩是客观存在，未起占有之私心，也不会有得失之痛苦。

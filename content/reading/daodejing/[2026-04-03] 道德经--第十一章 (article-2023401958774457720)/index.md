# 道德经--第十一章

> **Date / 日期**: 2026-04-03

---

**原文：**

```
三十辐共一毂，当其无，有车之用。

埏埴以为器，当其无，有器之用。

凿户牖以为室，当其无，有室之用。

故有之以为利，无之以为用。
```

**译文：**

```
三十根车辐横穿轮毂，中间留有空处，车轮才得以发挥转动的作用。

烧制陶土打造器皿，中间留出空处，器皿才能发挥盛放的作用。

开凿门窗建造房屋，有了门窗的空处，房屋才能发挥居住的作用。

所以，“有”使得万物产生效用，而“无”使得“有”发挥作用。
```

**个人体会：**

有之以为利，无知以为用。

除了举的例子之外，生活中还有很多：口袋，书包，瓶子...

能力地位，是有；可以拒绝不喜欢的人或事，是无；

挣钱攒钱，是有；花钱消费使钱体现出其价值，是无；

而且往往是先有了“无之以为用”，才会去想“有之以为利”，就是无中生有

人们是希望能够有居住的地方，居住的舒服，才会去建造房子；

人们是希望能够装东西提高效率，才会去设计发明口袋书包瓶子等工具；

没有人天生喜欢纸币或者账户上的余额，而是因为钱可以一定程度上买来想要的生活；

也没有人天生喜欢国央企公务员的光环，而是因为这些可以带来世俗意义的尊重和安全感；

老子的思想是其实是“无以为用”，从本源出发，再去思考如何有以为用。

现在的人是多追求“有以为利”，这是否倒果为因，我不知道。

但是如果觉得迷茫不知所从，老子的思想也许能帮你从杂乱的思绪中区分什么是以为在意实则可丢弃，什么是心之所向难以割舍。

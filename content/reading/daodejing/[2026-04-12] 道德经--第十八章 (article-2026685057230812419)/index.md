# 道德经--第十八章

> **Author / 作者**: 花雨迟  
> **Date / 日期**: 2026-04-12

---

**原文：**

```
大道废，有仁义；

智慧出，有大伪；

六亲不和，有孝慈；

国家昏乱，有忠臣。
```

**译文：**

```
大道被废弃了，才提倡仁义。

智慧巧诈出现了，就会产生大伪。

父子、兄弟、夫妻不和睦的时候，才会提倡孝慈；

国家混乱的时候，才会出现忠臣。
```

想起了高中写作文时很喜欢的一段排比句：

> “文王拘而演《周易》；仲尼厄而作《春秋》；屈原放逐，乃赋《离骚》；左丘失明，厥有《国语》；孙子膑脚，《兵法》修列；不韦迁蜀，世传《吕览》；韩非囚秦，《说难》《孤愤》；《诗》三百篇，大底圣贤发愤之所为作也。”

周朝礼崩乐坏之际，也是思想百家争鸣的时代。苦难的时代，才会有呕心沥血的作品。

也是反者道之动，弱者道之用的具象化体现。

当大环境越缺什么，就越需要什么，也越凸显这个稀缺的特质，就越有价值，真正的物以稀为贵。

同时，大环境越宣扬什么，代表着缺少什么。就像人性，自卑才炫耀，缺爱才花心。

当前居民存款屡创新高，国家不会鼓励大家存钱，而是引长期资本入市，势必要带动居民存款进入股市，所以这个时候提前把钱存入股市（优质标的），就是在等风来，明显的赚钱的机会，只不过以年为单位。

对于普通人来说，短线是不容易的，但大势是不会以分钟为单位进行的，所以识别大趋势，提前埋伏等风来，或者跟随大势做趋势，就是最稳妥保险的办法。

反过来想，飞鸟尽，良弓藏；狡兔死，走狗烹是否也体现了本章的思想呢？

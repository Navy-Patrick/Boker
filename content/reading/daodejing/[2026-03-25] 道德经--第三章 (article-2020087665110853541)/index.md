# 道德经--第三章

> **Author / 作者**: 花雨迟  
> **Date / 日期**: 2026-03-25

---

**原文：**

```
不尚贤，使民不争；

不贵难得之货，使民不为盗；

不见可欲，使民心不乱。

是以圣人之治，

虚其心，实其腹，弱其志，强其骨。

常使民无知无欲，使夫智者不敢为也。

为无为，则无不治。
```

**译文：**

```
不推崇有才德的人，不使老百姓相互争夺；

不珍贵难得的财物，不使老百姓心生偷盗；

不炫耀引起贪心的事物，使民心不迷乱。

因此，圣人的治理原则是：

让百姓保持内心的虚空，吃饱穿暖。减少功利心，增强百姓的筋骨体魄，

使老百姓去除智巧，没有贪欲。使得贪婪智巧之人不敢任意妄为，圣人按照无为的方式行事。

顺其自然，依循无为而治，那么天下就没有什么不能治理的了。
```

**个人体会：**

这章的内容我其实是不太理解的，或者说目前不是很认同。

文中讲不要举贤纳士，就不会使老百姓争夺权力。不去收藏珍贵的财物，老百姓不会心生偷盗，是人为的标准划分了界线，让人产生了羡慕嫉妒的情绪，进而引发混乱和争斗。这个期待是好的，但这样做是否能实现百姓不争民不为盗，我持怀疑态度。

而贤德有才能的人，金银珠宝天生就是稀缺产物，是人的主观使其珍贵，客观上来讲有才能的人和金银珠宝的稀少也是事实。我同意不应该去人为制造对立和攀比，但有才能的人可以治理好一个国家，如果不通过一些评价体系该如何选拔出对应的人才呢。

这篇写给君王的第三章是贯彻了前文的思想，但也许并没有给出君主治理国家的对策。古代成功的愚民策略，其实是有为，并不符合道家的无为概念，所以给出的治理国家策略在我看来也是十分勉强。

古代道家的无为而治和顺应自然在面对地主阶级产生了天然的排斥性，如果服务于君主，那么自己的理论无法自圆其说；如果不臣服于帝王，那么自己的理论就无法得到推广。所以道家在服务上层阶级是别扭且不自然的。

当然，也可能是我受到儒家文化浸润太久了，无法理解先人的思想。不过，历朝历代都没有解决[王朝周期论](https://zhida.zhihu.com/search?content_id=271975225&content_type=Article&match_order=1&q=%E7%8E%8B%E6%9C%9D%E5%91%A8%E6%9C%9F%E8%AE%BA&zhida_source=entity)，是否也是因为儒家的有为，导致上层阶级希望通过不断的控制穷人来维持自己优渥的统治地位，最后落了片白茫茫大地真干净。

# 道德经--第十七章

> **Author / 作者**: 花雨迟  
> **Date / 日期**: 2026-04-11

---

**原文：**

```
太上，不知有之；

其次，亲而誉之；

其次，畏之；其次，侮之。

信不足焉，有不信焉。  悠兮，其贵言。

功成事遂，百姓皆谓我自然。
```

**译文：**

```
最好的统治者，百姓并不知他的存在；

其次的统治者，百姓亲近而赞誉他；

再次的统治者，百姓畏惧他；  更次的统治者，人民轻蔑他。

统治者的诚信不足，百姓才不相信他。  最好的统治者悠闲自在，他很少发号施令。

当事情做成功了时，老百姓会觉得这就是本来的样子。
```

如果宇宙有神明的话，我想[塔](https://zhida.zhihu.com/search?content_id=272908848&content_type=Article&match_order=1&q=%E5%A1%94&zhida_source=entity)一定是最好的统治者，不知有之的状态。

可现实是，肉体凡胎很难不知其有，而比较选拔权衡是成为统治者的必经之路，在这个过程中已经不是无为了

众望所归民心所向之后还能不忘初心吗

百姓最喜欢的领导者是出身寒门不屈不挠，身居高位不忘初心

但现实不是爽文，更多的是贫困出身的挣扎在温饱线，贵人赏识的站队献忠诚，名利双收的已经听不到远方的哭声

屁股决定脑袋，在其位谋其政

从概率上基本锁死了统治者活成老百姓心目中的样子，此外统治者除了为人民服务，更多是考虑制衡和权术

即自身安全是第一要务

像老子讲的圣人，孔子讲的君子，[岳阳楼记](https://zhida.zhihu.com/search?content_id=272908848&content_type=Article&match_order=1&q=%E5%B2%B3%E9%98%B3%E6%A5%BC%E8%AE%B0&zhida_source=entity)中的“先天下之忧而忧，后天下之乐而乐”，

不一定是统治者，但是历史长河中的圣人君子

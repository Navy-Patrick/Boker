# 道德经--第五章

> **Date / 日期**: 2026-03-28

---

**原文：**

```
天地不仁，以万物为刍狗；

圣人不仁，以百姓为刍狗。

天地之间，其犹橐龠乎？

虚而不屈，动而愈出。

多言数穷，不若守于中。
```

**译文：**

```
天地并没有所谓的仁义慈悲，万物都像是草扎的祭品，在天地眼里并无特别。

圣人也是无所谓仁慈偏爱的，他对待百姓也像对待草扎的祭品一样，让百姓自作自息，顺其自然。

天地之间，不正像一个大风箱吗？

静止的时候，它只是一个空空如也的箱子，一旦运行起来，就会运转不息，永不穷尽。

大道恒久客观，言行干预反而导致势穷力屈，不如保持中立，守住内心的虚静。
```

**个人体会：**

有句话叫“时来天地皆同力，运去英雄不自由”，看到第五章中的“虚而不屈，动而愈出”，莫名想到这句话。

还有最近的新闻也引发人们关于健康与事业的讨论，老子的思想按照我个人的理解，工作应该劳逸结合，动静相宜，如果工作不顺利，那么可能是身体和天地气运不在一个频率，这时候需要做的是休养生息，等待蛰伏。等到身体精神都十分充沛，再去拼搏事业。

然而现实的舆论压力很难让个人完全的去放松休息，就像找工作如果gap了就跟留案底了一样，大部分人都是被社会的鞭子打着往前走，有的人心甘情愿，朝着理想前进；有的人苦不堪言，却依旧低头拉车，敢怒不敢言。

当讨论的声音越来越多，不同的观点也有足够多的拥趸，人们会越来越思考自己想要什么，渐渐为自己而活，我不知道这算不算好，但是相信有变化就是值得期待的事情。

以前有段时间很想去考公务员，后来才逐渐明白**人对稳定性的执念，是源于对未知的恐惧。**

而**确定性是普通人的精神鸦片**。但时代是不断变化的，把安全感寄托在一成不变的环境上，是最大的风险。这个世界唯一确定的就是变化，拥抱变化，才能确定的拥抱未来，才能在未来社会中占有一席之地。

万事万物一旦有了执念，就容易蒙蔽双眼，对事物就少了清晰客观的看法。

# Dual Platform Crawler：股票双平台内容爬虫看板

> 日期：2026-04-10
> 摘要：一个面向雪球 + 东方财富 的双平台股票评论爬虫项目，支持分页抓取、本地数据库存取，每日评论统计导出，双平台看板，图表交互
> 技术栈：Python / Requests / BeautifulSoup / Database / CLI
> GitHub：https://github.com/Navy-Patrick/dual-platform-crawler

![双平台数据看板](content\projects\dual-platform-crawler\双平台数据看板.png)
## 项目价值

该双平台情绪数据看板的核心价值在于填补了 “市场情绪量化观察” 的空白，为投资者提供了客观、多维度的情绪参考维度，具体体现在三方面：
- 解决单一平台情绪偏见问题：雪球（价值投资群体）与东方财富（短线交易群体）的投资者画像差异，形成 “长期价值情绪 + 短期交易情绪” 的互补视角，避免单一平台用户结构导致的情绪误判；
- 降低情绪决策的盲目性：通过历史评论数据的可视化呈现，将抽象的 “市场情绪” 转化为可追溯、可对比的波动曲线，帮助投资者识别情绪过热 / 过冷的拐点，减少情绪化交易；
- 构建个性化情绪观察体系：支持聚焦 “某几只股票” 的精准统计，满足投资者对持仓 / 关注标的的深度情绪跟踪需求，而非泛化的市场情绪数据，提升参考实用性。

## 核心亮点

- 双平台差异化互补
- 非预测性的理性定位
- 历史数据追溯与波动可视化

## 适用场景

- 持仓标的情绪跟踪
- 潜在买入/卖出情绪验证
- 不同投资风格情绪适配
